# PolityWatch AI

Political Transparency Dashboard powered by AI

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS + Recharts
- **Backend:** FastAPI (Python)
- **AI:** Cerebras API + Meta Llama
- **Deployment:** Docker

## Setup

### Prerequisites
- Node.js 18+
- Python 3.11+
- Docker & Docker Compose
